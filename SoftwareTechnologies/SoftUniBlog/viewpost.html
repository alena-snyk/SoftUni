Why choosing the right Storage Infrastructure

There are several toolchains involved in the automation process. Here’s a simplified example for the sake of illustrating concept: The developer checks the application code into a repository such as github. As soon as the code is committed, tools such as Jenkins automate the build and test process. Upon successful test execution, the application is moved to the “Staging” area before it is deployed into Production. Feedback is provided to the application development teams at every step of the way. This is the Continuous Integration (CI) process. Deployment into Production can also be automated where the application is constantly monitored, scaled and supported by the Ops teams, a process often referred to as Continuous Deployment (CD).
